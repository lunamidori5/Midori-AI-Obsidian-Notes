## Current Tasks

> [!example] Midori AI
> ```tasks
> path includes Development/Roadmap/Task List
> tag includes midori
> short mode
> show tree
> hide priority
> hide start date
> hide due date
> hide created date
> hide task count
> hide tags
> hide backlink
> hide edit button
> group by function task.tags.filter( (tag) => ! tag.includes("#midori") )
> sort by tag
> sort by priority
> sort by status
> ```
